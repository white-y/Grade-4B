# Day 8 记忆逻辑：服装店试穿！👗

## 今日单词
| 单词 | 中文 | 记忆钩子 |
|------|------|----------|
| T-shirt | T恤衫 | 👕 形状像字母"T"的短袖衫 |
| shirt | 衬衫 | 👔 shirt比T-shirt更正式，有领有扣 |
| skirt | 裙子 | 👗 s-kirt——"s"开头，像裙子旋转的样子 |
| shorts | 短裤 | 🩳 short(短的) + s = 短裤（裤腿短短的） |
| trousers | 长裤 | 👖 "trousers"——长长的词对应长长的裤子 |

## 联想故事 🎬
> 今天我们去服装店试穿衣服啦！
> 
> 👕 夏天穿**T-shirt**（T恤），因为它的形状像字母T！想想看T恤展开像不像一个T？
> 
> 👔 去学校面试穿**shirt**（衬衫），比T-shirt多了领子和扣子，更正式！
> 
> 👗 女孩子试穿**skirt**（裙子），旋转起来像花朵一样美！shirt和skirt很像，注意开头不同！
> 
> 🩳 夏天运动穿**shorts**（短裤），short就是"短"的意思，加s变成短裤！
> 
> 👖 冬天穿**trousers**（长裤），这个词好长——因为长裤也很长嘛！

## 记忆口诀 🎵
```
T-shirt像个T，
Shirt有领扣真整洁，
Skirt裙子转圈圈，
Shorts短裤短又短，
Trousers长裤暖过冬！
```

## 易混区分 ⚠️
| 对比 | 区别 |
|------|------|
| shirt vs skirt | sh-开头是衬衫，sk-开头是裙子 |
| shorts vs trousers | shorts短的，trousers长的 |
| T-shirt vs shirt | T-shirt休闲短袖，shirt正式有领 |
