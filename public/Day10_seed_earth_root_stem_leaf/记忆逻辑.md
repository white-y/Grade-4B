# Day 10 记忆逻辑：一粒种子的旅行！🌱

## 今日单词
| 单词 | 中文 | 记忆钩子 |
|------|------|----------|
| seed | 种子 | 🌰 "seed"发音像"西的"——西瓜的种子 |
| earth | 土地/地球 | 🌍 "earth"——大地妈妈，万物生长的地方 |
| root | 根 | 🌿 "root"像"肉特"——根像植物的肉，特别重要 |
| stem | 茎 | 🌾 "stem"像"死太木"——茎像一根直直的木头 |
| leaf | 叶子 | 🍃 "leaf"发音像"里夫"——叶子里负责光合作用 |

## 联想故事 🎬
> 让我们跟着一粒种子，看它怎样长大！
> 
> 🌰 一粒小小的**seed**（种子）被埋进了泥土里。
> 
> 🌍 **Earth**（土地）温暖又湿润，像妈妈的怀抱。
> 
> 🌿 种子先长出了**root**（根），就像小手一样扎进泥土里吸收营养。
> 
> 🌾 然后**stem**（茎）像一根小柱子一样从地面冒了出来，越长越高！
> 
> 🍃 最后茎的两边长出了绿绿的**leaf**（叶子），像小手掌朝太阳招手！

## 记忆口诀 🎵
```
Seed种子土里埋，
Earth大地妈妈怀，
Root长根往下扎，
Stem茎干往上爬，
Leaf叶子迎阳光来！
```

## 生长顺序记忆法 🌱→🌳
```
seed → root → stem → leaf
种子 → 生根 → 长茎 → 长叶
从下往上，从里到外！
```
