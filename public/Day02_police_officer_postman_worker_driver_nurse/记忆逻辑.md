# Day 2 记忆逻辑：上班早高峰！🌅

## 今日单词
| 单词 | 中文 | 记忆钩子 |
|------|------|----------|
| police officer | 警官 | 👮 police(警察) + officer(官员) = 警官，比police更正式 |
| postman | 邮递员 | 📮 post(邮件) + man(人) = 送邮件的人 |
| worker | 工人 | 🔧 work(工作) + er(人) = 工作的人 |
| driver | 司机 | 🚗 drive(开车) + r = 开车的人 |
| nurse | 护士 | 👩‍⚕️ "nurse"发音像"呐死"——护士说"别怕，不会死的！" |

## 联想故事 🎬
> 早上7点，城市开始热闹起来了！
> 
> 🚦 路口，**Police officer**（警官）穿着整齐的制服在指挥交通，他比普通police更有权威哦！
> 
> 📮 **Postman**（邮递员）背着大包，骑着绿色自行车到处送信——post是邮件，man是人，邮件人=邮递员！
> 
> 🏭 **Worker**（工人）戴着黄色安全帽走进工厂——work是工作，加er就是工作的人！
> 
> 🚌 **Driver**（司机）开着公交车载大家上班——drive是开车，加r变成开车的人！
> 
> 🏥 **Nurse**（护士）穿着粉色制服走进医院，温柔地照顾每一个病人。

## 记忆口诀 🎵
```
Police officer管交通，
Postman送信到门中，
Worker工厂干活忙，
Driver开车走四方，
Nurse医院爱心藏！
```

## 拆词小技巧 🔧
- **post + man** → 邮件 + 人 = 邮递员
- **work + er** → 工作 + 人 = 工人
- **driv(e) + er** → 开车 + 人 = 司机
- **police + officer** → 警察 + 官员 = 警官
- 记住：**-er** 和 **-man** 结尾表示"做某事的人"！
