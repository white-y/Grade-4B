# Day 17 记忆逻辑：真假与比较！💎

## 今日单词
| 单词 | 中文 | 记忆钩子 |
|------|------|----------|
| true | 真的 | ✅ "true"发音像"出儿"——出来的事实是真的 |
| lovely | 可爱的 | 💕 love(爱) + ly = 令人喜爱的 |
| special | 特别的 | ⭐ "special"像"死掰手"——特别厉害的人 |
| same | 相同的 | 👯 "same"只有4个字母，简简单单=一模一样 |
| thin | 瘦的 | 🐱 "thin"像"心"——瘦得只剩一颗心 |

## 联想故事 🎬
> 今天来玩"找不同"游戏！
> 
> ✅ 桌上有两颗钻石，一颗是真的，一颗是假的——老师问"哪个是**true**（真的）？"闪闪发光的那颗是真的！
> 
> 💕 小狗跑过来摇尾巴——好**lovely**（可爱）啊！love是爱，lovely就是令人爱的！
> 
> ⭐ 一堆普通蛋糕中间有一个彩虹色的——它好**special**（特别）！
> 
> 👯 两个双胞胎穿着一样的衣服——她们长得一模**same**（一样）！
> 
> 🐱 胖猫旁边有一只**thin**（瘦）猫，thin和thick(厚/胖)是反义词！

## 记忆口诀 🎵
```
True是真来假不要，
Lovely可爱人人夸，
Special特别不一般，
Same一样不分差，
Thin很瘦要吃饭！
```

## 对比记忆 ↔️
| 单词 | 反义/对比 | 记忆 |
|------|-----------|------|
| true | false | 真 ↔ 假 |
| same | different | 相同 ↔ 不同 |
| thin | fat/thick | 瘦 ↔ 胖 |
| special | ordinary | 特别 ↔ 普通 |
