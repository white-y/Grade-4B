# Day 11 记忆逻辑：种花任务！🌻

## 今日单词
| 单词 | 中文 | 记忆钩子 |
|------|------|----------|
| dig | 挖 | ⛏️ "dig"发音像"地格"——在地上挖格子 |
| plant | 种植/植物 | 🌱 "plant"既是动词"种"也是名词"植物" |
| sunflower | 向日葵 | 🌻 sun(太阳) + flower(花) = 追着太阳转的花 |
| field | 田野 | 🌾 "field"发音像"飞尔的"——田野里可以飞跑 |
| mountain | 山 | ⛰️ "mountain"里有"mount"(攀登)——要攀登的大山 |

## 联想故事 🎬
> 今天我们要完成一个种花任务！
> 
> ⛏️ 首先，拿起铲子**dig**（挖）一个小坑——dig, dig, dig！
> 
> 🌱 然后，把种子**plant**（种）下去——plant既是"种"也是"植物"！
> 
> 🌻 过了几周，长出了大大的**sunflower**（向日葵）！sun是太阳，flower是花，向日葵就是追着太阳转的花！
> 
> 🌾 **Sunflower**（向日葵）长在金色的**field**（田野）里，一望无际好美啊！
> 
> ⛰️ 远处还能看到高高的**mountain**（山），山顶上还有白白的雪呢！

## 记忆口诀 🎵
```
Dig挖土手不停，
Plant种下小生命，
Sunflower向阳开，
Field田野好宽大，
Mountain高山白云间！
```

## 拆词小技巧 🔧
- **sun + flower** → 太阳 + 花 = 向日葵🌻
- **mount + ain** → 攀登 + 后缀 = 山⛰️
- **dig → plant** → 先挖后种，动作顺序记忆！
