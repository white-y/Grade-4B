# Day 3 记忆逻辑：学校开放日！🏫

## 今日单词
| 单词 | 中文 | 记忆钩子 |
|------|------|----------|
| painter | 画家 | 🎨 paint(画画) + er(人) = 画画的人 |
| scientist | 科学家 | 🔬 science(科学) + ist(专家) = 科学专家 |
| writer | 作家 | ✍️ write(写) + r = 写作的人 |
| student | 学生 | 📚 "stu-dent"——"死读的"学生在学习 |
| keeper | 看守人 | 🔑 keep(保管) + er(人) = 保管的人 |

## 联想故事 🎬
> 今天是学校开放日，各行各业的人来学校展示他们的才艺！
> 
> 🎨 **Painter**（画家）戴着贝雷帽，在画布上画出了彩虹——paint是画画，加er就是画画的人！
> 
> 🔬 **Scientist**（科学家）穿着白大褂，做出了冒泡泡的有趣实验，同学们都惊呼"哇！"
> 
> ✍️ **Writer**（作家）坐在桌前专心写故事——write是写字，加r变成写字的人！
> 
> 📚 **Student**（学生）坐得端端正正，积极举手发言！
> 
> 🔑 **Keeper**（看守人）站在校门口，手里拿着一大串钥匙，守护学校的安全。

## 记忆口诀 🎵
```
Painter画画真好看，
Scientist实验冒泡泡，
Writer写出好故事，
Student举手回答妙，
Keeper看门保安好！
```

## 拆词小技巧 🔧
- **paint + er** → 画画 + 人 = 画家
- **writ(e) + er** → 写 + 人 = 作家
- **keep + er** → 保管 + 人 = 看守人
- **scien(ce) + tist** → 科学 + 专家 = 科学家
- 规律：**-er** 结尾的词大多表示"做某事的人"！
