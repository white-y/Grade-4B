# Day 4 记忆逻辑：家庭聚会做新衣！👨‍👩‍👦

## 今日单词
| 单词 | 中文 | 记忆钩子 |
|------|------|----------|
| dressmaker | 裁缝 | ✂️ dress(裙子) + maker(制作者) = 做裙子的人 |
| Mr | 先生 | 🤵 "Mr"是称呼男士的敬语，读"mister" |
| aunt | 阿姨/姑姑 | 👩 "aunt"发音像"安特"——安安全全的特别亲人 |
| uncle | 叔叔/舅舅 | 👨 "uncle"发音像"安口"——安慰你的口头禅叔叔 |
| boy | 男孩 | 👦 "boy"发音像"波伊"——活泼的小男孩 |

## 联想故事 🎬
> 过年了！一家人聚在一起，好热闹！
> 
> ✂️ **Dressmaker**（裁缝）正在缝纫机前做新衣服——dress是裙子，maker是做的人，合起来就是"做裙子的人"——裁缝！
> 
> 🤵 **Mr**（先生）Wang穿着帅气的西装，站起来给大家发红包说祝福语！
> 
> 👩 **Aunt**（阿姨）穿着漂亮的花裙子，提着一大盒蛋糕来了！
> 
> 👨 **Uncle**（叔叔）笑呵呵地进门，给每个小朋友拎了小礼物！
> 
> 👦 **Boy**（男孩）高兴得蹦蹦跳跳，拿着新年红包开心极了！

## 记忆口诀 🎵
```
Dressmaker做新衣，
Mr先生真有礼，
Aunt阿姨带蛋糕，
Uncle叔叔笑弯腰，
Boy男孩蹦蹦跳！
```

## 拆词小技巧 🔧
- **dress + maker** → 裙子 + 制作者 = 裁缝
- **Mr** → 是 Mister 的缩写，用在姓氏前面，如 Mr Wang
- **aunt/uncle** → 家庭成员词要整体记忆
- 想想你自己的 aunt 和 uncle 叫什么名字，会记得更牢！
