# Day 18 记忆逻辑：一天行动线！🎬

## 今日单词
| 单词 | 中文 | 记忆钩子 |
|------|------|----------|
| use | 使用 | 🔧 "use"发音像"优斯"——优秀地使用工具 |
| sleep | 睡觉 | 😴 "sleep"里有两个e——像闭着的两只眼睛 |
| dream | 做梦 | 💭 "dream"——d+ream(发音像cream奶油)——梦里有奶油城堡 |
| shout | 大喊 | 📢 "shout"发音像"烧特"——烧着了特别大喊 |
| should | 应该 | 💡 "should"里有would——should比would更强调"应该" |
| hit | 打/击 | 🏏 "hit"只有3个字母——快速有力，一击！ |

## 联想故事 🎬
> 一个小朋友的一天！
> 
> 🔧 早上起来，**use**（使用）牙刷刷牙、use筷子吃早饭！
> 
> 📢 上学迟到了，妈妈在后面**shout**（大喊）："快点！快点！"——注意不要和should混了！
> 
> 💡 老师说你**should**（应该）好好学习——should是"应该"！
> 
> 🏏 体育课上，**hit**（击打）棒球——hit! 一下就飞出去了！
> 
> 😴 晚上回到家，洗完澡就**sleep**（睡觉）了——两个e像闭着的眼睛！
> 
> 💭 睡着后开始做**dream**（梦），梦里有一座奶油城堡！

## 记忆口诀 🎵
```
Use使用各种工具，
Sleep睡觉眼睛闭，
Dream做梦到天际，
Shout大喊声音响，
Should应该记心里，
Hit一击力量大！
```

## 易混区分 ⚠️
| 对比 | 区别 |
|------|------|
| shout vs should | shout=大喊（t结尾）；should=应该（d结尾） |
