# Day 9 记忆逻辑：换装舞台走秀！🎭

## 今日单词
| 单词 | 中文 | 记忆钩子 |
|------|------|----------|
| scarf | 围巾 | 🧣 "scarf"发音像"是咔嗯夫"——冬天脖子"咔"一下围上 |
| sweater | 毛衣 | 🧶 sweat(出汗) + er——穿上会出汗的衣服 |
| dress | 连衣裙 | 👗 "dress"也可以当动词"穿衣服" |
| uniform | 制服 | 🎓 uni(统一) + form(形式) = 统一的穿着 |
| robe | 长袍 | 🥋 "robe"发音像"肉伯"——像肉一样包裹全身的袍子 |

## 联想故事 🎬
> 学校要举办换装走秀比赛！小朋友们轮流上台展示！
> 
> 🧣 第一个出场戴着**scarf**（围巾），软绵绵地围在脖子上，冬天好温暖！
> 
> 🧶 第二个穿着**sweater**（毛衣），好暖和——sweater里有sweat(出汗)，因为穿毛衣会热得出汗哦！
> 
> 👗 第三个穿着美丽的**dress**（连衣裙），在舞台上转了一圈好漂亮！
> 
> 🎓 第四个穿着整齐的**uniform**（制服），像个小军人一样帅——uniform=统一的形式！
> 
> 🥋 最后一个披着长长的**robe**（长袍），像魔法师一样酷！

## 记忆口诀 🎵
```
Scarf围巾暖脖子，
Sweater毛衣暖身子，
Dress裙子转圈子，
Uniform制服一样子，
Robe长袍像袍子！
```

## 拆词小技巧 🔧
- **sweat + er** → 出汗 + 的东西 = 毛衣（穿了会出汗）
- **uni + form** → 统一 + 形式 = 制服
- **dress** → 既是名词"连衣裙"，也是动词"穿衣"
