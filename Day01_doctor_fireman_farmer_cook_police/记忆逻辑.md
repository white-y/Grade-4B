# Day 1 记忆逻辑：职业小镇开门啦！🏘️

## 今日单词
| 单词 | 中文 | 记忆钩子 |
|------|------|----------|
| doctor | 医生 | 🩺 "doc"像"到客"——到客人家看病的人 |
| fireman | 消防员 | 🔥 fire(火) + man(人) = 灭火的人 |
| farmer | 农民 | 🌾 farm(农场) + er(人) = 在农场干活的人 |
| cook | 厨师 | 🍳 "cook"发音像"酷客"——很酷的做饭客人 |
| police | 警察 | 👮 "po-lice"发音像"泡利斯"——泡在街上守护你的人 |

## 联想故事 🎬
> 想象一个可爱的**职业小镇**，今天所有人都开始上班了！
> 
> 🩺 **Doctor**（医生）穿着白大褂，正在给小熊看病："张嘴说啊——"
> 
> 🔥 突然！**Fireman**（消防员）开着红色消防车飞奔而过——原来隔壁面包店着火了！
> 
> 🌾 田野那边，**Farmer**（农民）在笑呵呵地收白菜，farm就是农场，farmer就是农场里的人！
> 
> 🍳 面包店的火灭了，**Cook**（厨师）戴着高高的白帽子说："我来做蛋糕赔给大家！"
> 
> 👮 这时，**Police**（警察）走过来说："大家注意安全！"

## 记忆口诀 🎵
```
Doctor看病穿白衣，
Fireman灭火真勇气，
Farmer种地在田里，
Cook做饭香喷喷，
Police巡逻保安宁！
```

## 拆词小技巧 🔧
- **fire + man** → 火 + 人 = 消防员
- **farm + er** → 农场 + 做…的人 = 农民
- 记住：**-er** 和 **-man** 结尾常常表示"做某事的人"！
